## Sentence and structure conventions

- **Lead with the point.** Put the most important thing in the first sentence, not the last.
- **One idea per paragraph.** Long paragraphs lose readers.
- **Active voice by default.** "We shipped X" not "X was shipped."
- **No exclamation marks in business writing** unless celebrating a genuine win.

## Tone by context

| Context | Tone |
|---------|------|
| Cold outbound | Crisp, specific, brief — no fluff |
| Customer email | Warm, direct, helpful |
| Internal update | Outcome-focused, no theater |
| Executive summary | Bottom-line-up-front, no hedging |

## How to use this file

Any skill that writes external-facing text should check this file first. If a draft uses words from the "don't use" list, rewrite. If the structure buries the point, move it to the first sentence.
